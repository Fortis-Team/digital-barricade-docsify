<h1>Media Management</h1>
<div class="description">
    <p>       
        The uploaded media, which includes both photographs and videos, may easily be accessed by the distributor using the media library panel. The media that has been uploaded is utilized for creating a new playlist.
    </p>
    <img src="/images/distributor/medias/mediasTab.png" alt="uploaded_media_assets" width="100%" height="500">
</div>

<h1> Adding Media </h1>
<div class="description">
    <p>     
        The distributor can upload new media by clicking on the add media button and providing the desired name and media type (picture or video).
        To upload a new media, simply click the add media button.
    </p>
    <img src="/images/distributor/medias/mediasAdd.png" alt="add_media_assets" width="100%" height="200">
    <p>
        When the modal appears, the distributor can upload a new media after providing the desired name and importing a file (picture or video).
    </p>
    <img src="/images/distributor/medias/mediasAddModal.png" alt="add_media_assets" width="100%" height="500">
</div>

<br />
<h1>Edit Media</h1>
<div class="description">
    <p>      
        To update the uploaded media, simply select edit from the dropdown menu. The distributor is then able to change the media's name but is not allowed to modify the actual uploaded media.
    </p>
    <img src="/images/distributor/medias/mediasEdit.png" alt="edit_media_assets" width="100%" height="500">
    <img src="/images/distributor/medias/mediasEditModal.png" alt="edit_media_assets" width="100%" height="500">
</div>

<br />
<h1>Delete Media</h1>
<div class="description">
    <p>      
        To delete a media asset, click the delete option from the dropdown menu. A modal window to confirm the deletion of the media.
    </p>
    <img src="/images/distributor/medias/mediasDelete.png" alt="edit_media_assets" width="100%" height="500">
    <p>      
        Once the modal appears, the distributor can permanently delete the selected media asset.
    </p>
    <img src="/images/distributor/medias/mediasDeleteModal.png" alt="edit_media_assets" width="100%" height="500">
</div>
